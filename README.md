# 開発環境構築
- 必要な Node.js のバージョンは [.node-version](.node-version) を参照
- [Yarn](https://yarnpkg.com/ja/docs/install) をインストール

## アプリケーションの起動
- 下記手順で起動する

```
$ yarn
$ yarn start
```
